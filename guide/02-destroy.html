Get Dat

delete a dat

You should now be in the newly initialized dat repository from lesson 01.

Let's see how to deleting the dat repository and then re-initialize it again.

  1. Run dat destroy to delete the .dat folder.

  2. You should now have no .dat folder. You can check this with ls -a.

  3. Try to get the status by running dat status. You should get an error This is not a dat repository.

  4. Run dat init again. Now try to get the status again by running dat status. This time it should succeed!
